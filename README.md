# 🐍 Python Number Guessing Game

## 📌 Project Overview
This is my first Python mini project. It is a simple command-line Number Guessing Game where the player tries to guess a randomly generated number within a limited number of attempts.

## ✨ Features
- Three difficulty levels (Easy, Medium, Hard)
- Random number generation
- Limited attempts
- Score calculation
- Input validation
- Play Again option
- User-friendly command-line interface

## 🛠️ Technologies Used
- Python 3

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the `number_guessing_game.py` file.
4. Choose a difficulty level.
5. Guess the correct number before your attempts run out.
