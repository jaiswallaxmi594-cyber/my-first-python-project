# 🐍 Python Quiz Game

## 📌 Project Overview
This is my first Python mini project. It is a simple command-line quiz game where users answer multiple-choice questions and receive a final score based on their correct answers.

## ✨ Features
- Multiple-choice quiz questions
- User-friendly command-line interface
- Instant feedback for each answer
- Automatic score calculation
- Final performance message

## 🛠️ Technologies Used
- Python 3

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the `quiz_game.py` file.
4. Enter your answers using A, B, C, or D.
